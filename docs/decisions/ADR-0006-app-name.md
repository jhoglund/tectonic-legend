# ADR-0006: App name

**Date:** 2026-05-14
**Status:** Proposed
**Source:** [docs/market-research.md §5](../market-research.md)

## Context

"Tectonic" is generic and not trademark-defensible for puzzle apps. Keesing owns the puzzle-app association in the App Store (4,900 ratings, top result). A search for "Tectonic" returns Keesing first; a player who hears "try Tectonic" lands on a competitor by default. We need a name that wins the app-store search for our own players while still riding "Tectonic" / "Suguru" as discovery keywords.

The repo name (`tectonic-for-the-win`) is a placeholder and does not need to be the brand.

## Decision (proposed)

Adopt a **distinctive primary brand** with "Tectonic Puzzles" (or "Suguru") as the subtitle / ASO keyword string. Candidates to evaluate (placeholder list — Jonas to revise):

- *Cage* — short, evocative, names the core mechanic.
- *Nikoli* — historical, but trademark-encumbered (Nikoli is a real Japanese publisher). **Strike.**
- *Sumi* — Japanese for "corner / nook," resonates with the cage shape; short.
- A coined name (think *Calm*, *Headway*, *Otter*) — fully ownable, expensive to build awareness.

Subtitle: `Tectonic & Suguru Puzzles` to capture ASO without committing the brand to either generic.

## Consequences

- **Enables:** trademark filing; clean ASO positioning ("the next puzzle after Sudoku" with a distinctive brand); long-term defensibility.
- **Precludes:** continuing to use "Tectonic" as the primary brand without legal review.
- **Implies:** trademark search before any App Store submission; logo + wordmark designed in Open Design once the name is picked; domain acquisition; the repo can stay as `tectonic-for-the-win` (working title) or be renamed once the brand is set.
